SPRITE TEMPLATE
16-BIT GRID

Blank grids for hand-designing characters. Screenshot, open in any paint app, color the squares, send it back. Every grid here uses the exact same cell size as our in-game sprites — your drawings drop straight in.

▸ HOW TO USE

  1. Pick a grid based on what you’re drawing — character, item, or boss.
  2. Screenshot the grid (or right-click → save image).
  3. Open it in any paint app — iPhone Markup, Preview, Photoshop, MS Paint, whatever.
  4. Color cell by cell. Use the palette below, or make up your own colors.
  5. Send it to me — I’ll convert it to an SVG sprite and add it to the game.
