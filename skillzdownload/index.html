What Already Exists

  • Anthropic created the SKILL.md open standard (Dec 2025) — adopted by OpenAI, Microsoft, and open-source agents
  • Installation is one folder, one file — the format is simple and universal
  • LobeHub indexes 170K skills, SkillsMP claims 700K — all free, all unvetted
  • Only one paid competitor (SkillDepot, launched April 2, 2026 — 3,800 skills)

What Doesn't Exist

  • A consumer-friendly marketplace anyone can browse
  • Subscription model with versioned updates — no platform has this
  • Creator profiles with brands, followers, and revenue
  • Security — 12% of skills on one platform were found malicious
